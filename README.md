# Clean-Windows-Install (PT-BR) 
Uma instalação limpa de Windows, precisa ter arquivo .ISO para fazer o boot!

Para instalar o Windows a partir de um DVD

    Copie o arquivo autounattend.xml para a pasta raiz de um pen drive e conecte-o ao computador.
    Inicialize a partir da unidade de DVD para iniciar a instalação automática.

Para instalar o Windows a partir de um pen drive

    Copie o arquivo autounattend.xml para a pasta raiz do pen drive que contém os arquivos de instalação do Windows e conecte-o ao computador. (O arquivo autounattend.xml estará localizado na mesma pasta que o setup.exe.)
    Inicialize a partir do pen drive para iniciar a instalação automática.
